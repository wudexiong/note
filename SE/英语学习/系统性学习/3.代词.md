# 代词

避免重复而用来代替名词的词语。


## 人称代词

| 主格          | 宾格         |
| ------------- | ------------ |
| 我 (I)        | 我们 (we)    |
| 你 (you)      | 你/你们 (you) |
| 她 (she)      | 她 (she)     |
| 他 (he)       | 他 (he)      |
| 它 (it)       | 它/他们/她 (it) |
| 我们 (we)     | 我们 (us)    |
| 他们/她 (they) | 他们/她们 (them) |


人称代词主格单数作并列主语的顺序: you he(she) and I
人称代词主格复数作并列主语的顺序: we you and they
